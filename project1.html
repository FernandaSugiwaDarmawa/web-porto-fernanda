Project Web: Academic Study Planner

This project is a project taken from a problem at Palangkaraya University, especially the Faculty of Law, where there is still no online KRS system.

Main Features:

  • Study Plan Card Registration.
  • Study Plan Card Management.
  • Study Plan Card Archive.

This project was developed using Express.js and React, react was chosen because it is for building user interfaces (UI), especially for web-based applications with interactive and dynamic interfaces. This project aims to address a significant problem at the Faculty of Law, Palangkaraya University, where there is currently no online system for course registration (KRS). This project will create an efficient and streamlined online KRS system, allowing students to register for courses more easily and accurately, reducing administrative burden, and improving the overall academic experience at the faculty.
