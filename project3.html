Project Web: Data Manivest

This project is a project belonging to PT Telkom Data Ekosistem where this website was created to manage all data from Telkom Data Ekosistem customers. So that this website makes it easier for companies to manage their customer data, this project uses laravel 12 framewrok.

Main Features:

  • Create User.
  • Add Customers.
  • Export Excel.
  • Edit Data.
  • Delete Data.
