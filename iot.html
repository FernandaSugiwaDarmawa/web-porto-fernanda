Project IoT: Passenger Seat Availability Monitoring Tool for Jenggala Trains

This project aims to develop a monitoring tool for passenger seat availability in Jenggala trains, providing real-time information about seat occupancy and availability.

Main Features:

  • Real-time seat availability updates.
  • Integration with train scheduling systems.
  • Train presence tracking.

The technologies used in this project include seat occupancy sensors, communication modules, and integration with a web-based dashboard for real-time updates. This project is made using esp32 microcontroller and using antares platform to store data and using laravel framework in developing the monitoring website.
