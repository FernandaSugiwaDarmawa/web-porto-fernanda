Project Web: SKILLKITA

A website that contains training and job search in the 4.0 era, the training is also adapted to the times so that job seekers can have the skills needed in era 4.0.

Main Features:

  • Job Training.
  • Job Vacancies.
  • Certification.

The technologies used in this project include HTML, CSS, and JavaScript for the frontend, along with a responsive design to ensure accessibility across devices. The backend is powered by the Laravel framework, providing robust support for user authentication, job postings, and training management. Additionally, MySQL is used as the database to store user data, training courses, and job listings, ensuring efficient data retrieval and management. The site is hosted on a cloud platform to ensure scalability and high availability.
